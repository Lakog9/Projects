# Project 1 — Drug-Drug Interaction (DDI) Prediction with kNN

Predicts DrugBank DDI relation type for a drug pair using nearest-neighbor similarity between molecular features.

## Features / distances
- Morgan fingerprints → Tanimoto (Jaccard) distance
- RDKit2D descriptors → cosine distance

## Data
Uses DDI-Bench data (download separately; not included in this repo).
